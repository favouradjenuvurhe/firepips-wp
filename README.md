## 🚀 Firepips WP SMTP

Firepips WP SMTP is a powerful and lightweight SMTP plugin for WordPress that ensures reliable email delivery, logging, and GitHub-based auto updates.

---

## ✨ Features

- Easy SMTP Configuration  
- Send Test Email  
- Email Logging (Sent & Failed)  
- Error Tracking for Debugging  
- GitHub Auto Updates  
- Lightweight & Developer Friendly  

---

## 📦 Installation

### Method 1: Upload Plugin
1. Download the plugin ZIP  
2. Go to WordPress Dashboard → Plugins → Add New  
3. Click Upload Plugin  
4. Activate the plugin  

---

### Method 2: Manual Installation
1. Extract the plugin folder  
2. Upload to:
   /wp-content/plugins/firepips-wp
3. Activate from WordPress dashboard  

---

## ⚙️ Setup

1. Go to Firepips SMTP in your WordPress admin  
2. Enter your SMTP details  
3. Save settings  
4. Send a test email
